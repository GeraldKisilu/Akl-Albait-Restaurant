## How to Replace Images

1. **Copy** your new image URL
2. **Paste** it into the **"New Image URL"** column beside the corresponding item
3. **Then edit `src/app/App.tsx`** — find the image reference inside the `IMG` object or the `ITEMS` array and replace the URL

> 💡 **Tip:** Most image references are mapped through the `IMG` object (around line 60-160). If you replace the URL there, it updates everywhere that item appears (menu page, highlights, basket, etc.).

> For items using `IMG.someKey`, find the key in the `IMG = { ... }` section.
> For items with inline URLs (like Meals & Deals), scroll down to `DEAL_ITEMS` and update them there.
